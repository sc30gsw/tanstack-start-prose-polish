## カテゴリ凡例

- `ECC/メタツール`: ECC 自体の案内、インストール、スキル棚卸し、hooks、instinct、コンテキストやコスト管理。
- `計画/設計/アーキテクチャ`: 実装前の計画、構造設計、ADR、プロダクト要求から設計への変換。
- `コード品質/レビュー/リファクタ`: コードレビュー、保守性、コメント、型設計、デッドコード整理。
- `ビルド復旧`: コンパイル、型チェック、依存関係、ビルド失敗の復旧。
- `テスト/検証/品質ゲート`: TDD、E2E、カバレッジ、品質ゲート、リリース前検証。
- `フロントエンド/UI/デザイン`: UI 実装、アクセシビリティ、SEO、モーション、ブラウザ QA、フロントエンド系フレームワーク。
- `バックエンド/API/データ`: API、DB、ORM、サーバーサイド、データアクセス、バックエンド設計。
- `言語/フレームワーク別`: 特定言語やフレームワーク固有の実装・レビュー・テスト。
- `セキュリティ/コンプライアンス`: 脆弱性、認証認可、秘密情報、PHI/HIPAA、規制・安全性。
- `DevOps/インフラ/ネットワーク`: Docker、デプロイ、ネットワーク、homelab、運用基盤。
- `AIエージェント/自律化`: 自律ループ、マルチエージェント、評価ハーネス、AI 実行基盤。
- `ドキュメント/ナレッジ/リサーチ`: ドキュメント更新、最新情報調査、ナレッジ管理、コードベース理解。
- `GitHub/プロジェクト運用/セッション`: PR、セッション保存、チェックポイント、プロジェクト運用。
- `ビジネス/オペレーション`: 請求、物流、投資家対応、調達、メール、営業、業務運用。
- `メディア/コンテンツ`: 記事、ブランドボイス、動画、スライド、画像・音声生成、SNS。
- `科学/データ/ML`: 機械学習、推薦、学術文献、特許、バイオ系データベース。
- `その他/専門領域`: 上記に明確に入らない専門用途。
